# 🌤 Weather Notes

A modern iOS app that lets you create notes with real-time weather data based on your location.

Built with **SwiftUI + MVVM**, using clean architecture and a minimal, iOS-style UI.

---

## ✨ Features

- 📝 Create notes with text  
- 🌍 Automatic location detection  
- ☀️ Weather integration (temperature, condition, icon)  
- 🌙 Dark mode support  
- 💾 Local storage with CoreData  
- 📱 Clean UI inspired by iOS Notes  

---

## 🛠 Tech Stack

- SwiftUI (iOS 17)  
- MVVM architecture  
- CoreData  
- URLSession (network layer)  
- OpenWeather API
